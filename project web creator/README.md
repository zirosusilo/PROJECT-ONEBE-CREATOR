### Metode 1: Menggunakan Batch File (Paling Mudah)
1. Double-click file **`start-server.bat`**
2. Website akan otomatis terbuka di browser pada `http://localhost:8000`

### Metode 2: Menggunakan Python
```bash

python -m http.server 8000

# Python 2
python -m SimpleHTTPServer 8000
```
Kemudian buka browser ke: `http://localhost:8000`

### Metode 3: Menggunakan VS Code Live Server
1. Install extension "Live Server" di VS Code
2. Right-click pada `index.html` dan pilih "Open with Live Server"

### Metode 4: Menggunakan Node.js (Jika Terinstall)
```bash
npm install
npm run dev
```
